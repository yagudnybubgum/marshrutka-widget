# Виджет расписания маршруток

Сайт с виджетом ближайших маршруток для Янино-1: маршруты 533, 429, 664, 430А, 453 и сводка отправлений с Ладожской.

## Установка

```bash
npm install
```

## Запуск

```bash
npm run dev
```

Откройте браузер по адресу, который покажет Vite (обычно http://localhost:5173).

Перед запуском и сборкой Excel-файлы из `public/schedule-*.xlsx` автоматически конвертируются в `public/schedules/*.json`.

## Использование

1. Положите файлы `schedule-{routeId}.xlsx` в папку `public/`
2. Запустите `npm run convert-schedules` (или `npm run dev` / `npm run build`)
3. Виджет автоматически подтянет расписание и покажет ближайшие рейсы
4. Время до рейса обновляется каждую секунду

## Формат файла

Файл XLSX должен содержать:
- Строку с периодами (будние/выходные) или строку с направлениями
- Колонки с временем отправления (формат HH:MM, HH.MM или Excel-время)
- Направления определяются по ключевым словам в заголовках колонок

## Скрипты

- `npm run dev` — локальная разработка
- `npm run build` — production-сборка
- `npm run convert-schedules` — конвертация xlsx → json
- `npm test` — unit-тесты
- `npm run lint` — ESLint

## Особенности

- Автоматический выбор буднего/выходного расписания с учётом праздников РФ
- Показ ближайших рейсов и полного расписания по маршрутам
- Таб «С Ладожской» — сводка отправлений нескольких маршрутов
- Адаптивный дизайн для мобильных устройств
- PWA manifest для добавления на главный экран
