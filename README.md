# Виджет расписания маршруток

Небольшой сайт с виджетом для отображения ближайших маршруток в двух направлениях.

## Установка

```bash
npm install
```

## Запуск

```bash
npm run dev
```

Откройте браузер по адресу, который покажет Vite (обычно http://localhost:5173)

## Использование

1. Загрузите файл расписания в формате XLSX через кнопку загрузки
2. Виджет автоматически определит колонки с расписанием в двух направлениях
3. Отобразятся ближайшие 3 рейса в каждом направлении с подсветкой ближайшего

## Формат файла

Файл XLSX должен содержать:
- Первая строка - заголовки колонок
- Колонки с временем отправления (формат HH:MM или HH.MM)
- Поддерживается автоматическое определение направлений по ключевым словам: "туда", "обратно", "направление 1/2"
- Если ключевые слова не найдены, используются первые две колонки с данными

## Особенности

- Автоматическое определение направлений
- Показ ближайших 3 рейсов в каждом направлении
- Подсчет времени до ближайшего рейса
- Автоматическое обновление каждую минуту
- Адаптивный дизайн для мобильных устройств

## Деплой на GitHub Pages

Проект настроен для автоматического деплоя на GitHub Pages через GitHub Actions.

### Настройка

1. Убедитесь, что в настройках репозитория включен GitHub Pages:
   - Settings → Pages → Source: "GitHub Actions"

2. Если имя репозитория отличается от `marshrutka-widget`, обновите `base` в `vite.config.js`:
   ```js
   base: process.env.GITHUB_PAGES === 'true' ? '/ваше-имя-репозитория/' : '/',
   ```

3. Закоммитьте и запушьте изменения в ветку `main` или `master`

4. GitHub Actions автоматически соберет и задеплоит проект

Сайт будет доступен по адресу: `https://ваш-username.github.io/marshrutka-widget/`
